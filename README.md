# FoodieHub 🍔

A modern, responsive, and dark-themed Food Delivery web application inspired by platforms like Zomato and Swiggy.

FoodieHub delivers a premium user experience with glassmorphism UI, interactive animations, restaurant filtering, dynamic cart management, and responsive layouts optimized for desktop, tablet, and mobile devices.

## ✨ Features

### 🎨 Modern UI/UX
- Premium orange-red dark theme
- Glassmorphism-based interface
- Smooth animations and hover effects
- Responsive layout for all devices

### 🔍 Smart Restaurant Discovery
- Restaurant and cuisine search
- Location-based suggestions
- Rating and dietary filters
- Interactive browsing experience

### 🛒 Dynamic Cart System
- Real-time cart updates
- Quantity management
- Auto-calculated pricing
- Coupon discount support
- Persistent cart using LocalStorage

### 🚚 Interactive Order Tracking
- Live order success modal
- Animated delivery tracking
- Real-time delivery status timeline
- Delivery partner information panel

### 📱 Mobile Responsive Design
- Mobile-first optimization
- Sliding mobile navigation drawer
- Tablet and desktop responsive layouts
- Smooth adaptive UI transitions


## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- LocalStorage API
